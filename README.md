# UFlash Super-App Proposal & Prototype

This repository contains the proposal and AI-driven safety engine for **UFlash**, a campus-exclusive Super-App created for an in-class assignment in the **Professional Environments** module.

---

## 📄 Proposal & Documentation
* **Project Proposal:** Download the attached `UFlash_Proposal.pdf` in this repository for full details on system architecture, crowdsourced mapping, and financial feasibility.

---

## 🤖 AI Studio Prototype (AI Allergy Guard)
The **AI Allergy Guard** module uses the Gemini API via Google AI Studio to screen canteen menu ingredients against user allergy profiles in real-time.

* **Live AI Studio Model/App Link:** [https://ai.studio/apps/a6fe8aaa-c57f-43b8-b0c8-e73792b42425]
* **Tech Stack:** Python, Google AI Studio (Gemini API)

---

## 🛠️ Key Features Summary
* **Smart Dining:** Digital token queueing (`Flash-Queue`) to save ~20 mins/day.
* **Health Safety:** AI-powered allergen screening (`AI Allergy Guard`).
* **Navigation:** Crowdsourced campus GPS layer (`Smart Maps`).
